Python Port Scanner Details

This Python-based port scanner tool was developed to efficiently identify open network ports on a specified target host or IP address. The tool leverages the `socket` library to establish connections and determine the status of different ports. The `ipaddress` module is used for validating IP address inputs, and basic multi-threading using the `threading` library was implemented to improve scanning speed by concurrently checking multiple ports.

  • Scans a range of TCP ports on a given IP address or hostname.

  • Utilizes the `socket` library for network connection attempts.

  • Includes basic multi-threading for faster scanning.

  • Provides clear output indicating open and closed ports.
